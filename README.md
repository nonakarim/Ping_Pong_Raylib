# 🏓 Ping Pong (C++)

A classic Pong game implemented in C++ using object-oriented programming.

## 📌 Features
- Paddle movement
- Ball physics and collision detection
- Score system
- Sound effects
- Menu system
- Clean modular structure (.h / .cpp separation)

## 🛠 Technologies Used
- C++
- Object-Oriented Programming
- Makefile build system

## 🚀 How to Build & Run

```bash
make
./main
